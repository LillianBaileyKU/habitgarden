# The Habit Garden
Turn your daily habits into a growing virtual garden!
Habit Garden is a small mobile app built in Unity that helps users stay consistent with their habits by turning them into growing plants.

## Features
* Add and manage up to 3 habits
* Track progress and setbacks
* Visual plant growth based on consistency
* Persistent saving

## Requirements
* None! This is an android app!

## How to Install
1. Download TheHabitGarden.apk
2. Run the .apk on your Android phone to install the app!

## How to Play
1. Press "Add Habit" to create a new habit
2. Use the plus and minus buttons to track progress and setbacks
3. Watch your plants grow as you stay consistent
4. Delete habits anytime with the "Delete Habit" button
